VS CODE 단축키 정리

  • ctrl + space : 자동 완성 기능
  • alt + shift + f : 자동 정렬 기능
  • ctrl + x : 한 줄 자르기
  • ctrl + alt : 방향키 : 여러 군데에 한 번에 적기
  • ctrl 누르고 클릭을 여러 군데에 하면 여러 군데에 한 번에 적을 수 있다.
    CSS 내용 정리
  • px은 절대 크기 단위
  • em, rem은 글씨체에 관한 상대 크기 단위

    HTML 태그 비교

  • p 태그 : 단락 구분 o
  • span 태그 : 단락 구분 x
